LEIA ANTES DE USAR

O Projeto Workspace é dividido em duas partes: a página web e a aplicação desktop. Ao abrir a pasta Workspace, o projeto da página web está na pasta web_page, e o código principal é o app.py; enquanto o projeto do software está na pasta desktop_app, e o código principal é o main.py.

Banco de dados (MySQL)

Para que você possa rodar os códigos das aplicações, será necessário ter o MySQL instalado e abrir o terminal do Prompt de Comando. Em seguida, execute primeiramente o seguinte:

"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -p

Depois, digite a senha do seu MySQL e rode isto:

SOURCE schema.sql;

Após isso, clique no arquivo .env e preencha as informações referentes ao host, user, password e ao nome do banco de dados (neste caso, o nome é workspace_db).

Instalar Bibliotecas

Antes de realizar os testes, abra o terminal, localize a pasta onde está o arquivo requirements.txt (neste caso, na pasta Workspace) e execute o seguinte:

pip install -r requirements.txt

Teste Final

Pronto, após seguir esses passos acima, você já pode testar o projeto.

Site — Para testar a página web, abra a pasta web_page e execute o código app.py. Em seguida, clique no link exibido no terminal da sua IDE. Geralmente, esse link é: http://127.0.0.1:5000/

Software — Para testar o software, basta abrir a pasta desktop_app e executar o código main.py.
